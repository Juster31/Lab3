# Лабораторная работа 3
#### Симулятор сельского хозяйства

## Запуск локально
1. Склонируйте репозиторий или скачайте файлы проекта.
2. Откройте index.html в современном браузере (Chrome/Edge/Firefox).

## Структура проекта
- index.html - основной HTML-документ.
- style.css — стили для графического
- script.js - скрип, управляющий игрой
- classes.js - скрип, содержащий определение классов
- - Cell - класс клетки
- - - Water - класс воды, наследуется от Cell, создает влажность
- - - Earth - класс зелми, наследуется от Cell, реагирует на клетки Воды
- - Plant - класс растения
- - - Cactus - класс кактуса, наследуется от Plant, требуемая влажность 30-0
- - - Swater - класс болотника, наследуется от Plant, требуемая влажность 100-70
- - - Potato - класс воды, наследуется от Plant, требуемая влажность 40-80

## процесс игры
1. Лопата - позволяет выкапывать растения
2. Ведро - позволяет забирать и выливать воду в клетку
3. Семена - позволяют посадить растение в клетку Земли
4. Состояние растений меняется каждые 3 секунды, либо они растут, либо умирают

## Визуализация
- цвет земли меняется от цвета песка до черного, в зависимости от влажности
- растения отображаются картинками,а под ними их состояние. 0 - только создали, 1-3 стадии роста. Dead - растение умерло
